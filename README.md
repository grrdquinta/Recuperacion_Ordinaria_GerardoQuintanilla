Escuelita Marvel App
Una aplicación web moderna desarrollada con React que ofrece una experiencia educativa interactiva inspirada en el universo Marvel. 
La aplicación incluye un sistema de navegación intuitivo, dashboard administrativo y gestión de estudiantes.

🚀 Características

Dashboard Interactivo: Panel de control con métricas y estadísticas
Gestión de Estudiantes: Página dedicada para administrar información estudiantil
Navegación Intuitiva: Barra de navegación responsive y moderna
Diseño Responsive: Adaptable a diferentes tamaños de pantalla
Interfaz Moderna: Diseño atractivo con temática Marvel

🛠️ Tecnologías Utilizadas

React 19.1.0: Framework principal para la construcción de la interfaz
React Router DOM 7.6.3: Manejo de rutas y navegación
React Hook Form 7.60.0: Gestión de formularios eficiente
Tailwind CSS 4.1.11: Framework CSS para estilos modernos
Lucide React 0.525.0: Iconografía moderna y consistente
Vite: Herramienta de build rápida y moderna

📦 Instalación
Prerrequisitos

Node.js (versión 18 o superior)
npm o yarn

Pasos de instalación

Clonar el repositorio
bashgit clone https://github.com/tu-usuario/escuelita-marvel-app.git
cd escuelita-marvel-app

Instalar dependencias
bashnpm install

Ejecutar en modo desarrollo
bashnpm run dev

Abrir en el navegador
http://localhost:5173
